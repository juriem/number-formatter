Number Formatter
=========

A small library that adds commas to numbers

## Installation

`npm install juri.em/number-formatter`

## Usage 
    
    var NumFormatter = require('juriem/nnumber-formatter'); 
    
    var formettedNum =  numFormatter(35666); 
    
    Output should be `35,666`
